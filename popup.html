Setup Required

Prerequisites: Chrome 138+ on Windows/macOS/Linux, 22GB+ free disk space, and either GPU >4GB VRAM or 16GB RAM with 4+ CPU cores.

To use this extension:

  1. Open chrome://flags/#optimization-guide-on-device-model and set to Enabled
  2. Open chrome://flags/#prompt-api-for-gemini-nano and set to Enabled (or try #prompt-api-for-gemini-nano-multimodal-input)
  3. Click Relaunch to restart Chrome
  4. Open chrome://components, find Optimization Guide On Device Model, click Check for update
  5. Open chrome://on-device-internals and check the Model Status tab for errors
  6. Wait for the model to download (~22GB)
