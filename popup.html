Setup Required

Prerequisites: Chrome 138+ on Windows/macOS/Linux, 22GB+ free disk space, and either GPU >4GB VRAM or 16GB RAM with 4+ CPU cores.

To use this extension:

  1. Open chrome://flags/#optimization-guide-on-device-model and set to Enabled
  2. Open chrome://flags/#prompt-api-for-gemini-nano-multimodal-input and set to Enabled
  3. Restart Chrome
  4. Wait for Gemini Nano to download (~22GB)
